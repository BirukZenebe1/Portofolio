# Biruk Zenebe Portfolio

Personal portfolio website focused on AI and Machine Learning projects, deployed on Vercel.

## Live Site

[birukzenebe.vercel.app](https://birukzenebe.vercel.app)

## Highlights

- Modern responsive single-page portfolio
- Light/Dark mode toggle
- Animated project and experiment cards
- Direct links to GitHub, LinkedIn, and email

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Lucide Icons
- React Parallax Tilt

## Run Locally

```bash
npm install
npm start
```

## Build For Production

```bash
npm run build
```

## Deployment

This repo is connected to Vercel and deploys from the `main` branch.
